# neurodev_back

## Устанавливаем пакеты с зависимостями 

```npm install```

## Запускаем файл client.js

```node client.js```

## Запускаем Web-Socket server 

```node ws.js```

## Запускаем сервер, обрабатывающий get и post запрпосы

```node server.js```
